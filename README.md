# Bunkers Inc. — Web oficial

Sitio web del videojuego **Bunkers Inc.**, un roguelike de supervivencia post-apocalíptico desarrollado por **2HN Games** como parte de la asignatura Proyectos 2 del Grado en Desarrollo de Videojuegos (UCM).

🌐 **[Ver sitio en vivo](https://rluy02.github.io/Bunkers-Inc/)**

🎮 **[Jugar en itch.io](https://julsv.itch.io/bunkers-inc)**

---

## Sobre el juego

Tras un evento nuclear global, el exterior está contaminado por radiación y el búnker en el que vives empieza a fallar. Alterna entre la gestión de tu base y expediciones procedurales al exterior para reunir recursos, mejorar tu arma y fabricar el traje antirradiación antes de que sea tarde.

- **Género:** Roguelike · Supervivencia · Gestión de recursos
- **Vista:** 2D top-down, pixel-art
- **Plataformas:** Windows 10/11 (64-bit)
- **Controles:** Teclado y ratón / Mando (PS5, Xbox)

## Sobre este repositorio

Este repositorio contiene únicamente el **código fuente de la web** del juego. El código fuente del juego en sí vive en un repositorio privado de la organización académica.
