Como comentar el código ​

Para comenta el código se debe hacer en el .h. El estilo de comentar se asemeja mucho al JSDoc.

La forma de comentar es con ///:

c
/// @brief Descripcion corta
/// @details Detalles más profundos
/// No es necesario ponerlo todo en una linea.
/// Se puede utilizar tantas como necesiteis
/// @param x descripcion del parámetro x
/// @param y descripcion del parámetro y
/// @return z descripcion del valor devuelto z
int sumOfVecotor(std::vector<int> x)

Se puede introducir bloque de código estilo Markdown, que luego se puede ver muy fancy, pero en principio es simplemente la forma de arriba.
